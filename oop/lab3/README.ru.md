# Лабораторная 3 — Сериализация объектов (вариант 3, текстовый формат)

Порядковый номер в группе: **12**.
Вариант: `(12 mod 5) + 1 = 3` → текстовый формат.

## Предметная область

Иерархия моделирует наземный, водный и воздушный транспорт. Девять
конкретных классов наследуются от трёх промежуточных, у которых, в
свою очередь, общий базовый класс `commonBase` с полями
идентификации:

```
Vehicle (интерфейс)
├── Land
│   ├── Car
│   ├── Motorcycle
│   └── Truck
├── Water
│   ├── Boat
│   └── Ship
└── Air
    ├── Airplane
    └── Helicopter
```

Итого **6 + 3 = 9 классов** (без учёта двух абстрактных, которые
требование уже считает достаточными).

## Открытость к расширению

Чтобы добавить новый класс, нужны три шага и **никаких** правок
существующего кода:

1. Объявить структуру, встроенную (`embed`) в одну из групповых баз
   (`landBase` / `waterBase` / `airBase` или `commonBase`).
2. Реализовать четыре метода интерфейса `Vehicle`. В `Fields()` берём
   список дескрипторов базы и дописываем свои.
3. Зарегистрировать фабрику в `init()` своего пакета.

В проекте **нет ни одного** `if` / `switch` / рефлексии, который
ветвился бы по конкретному классу — ни в сериализаторе, ни в
редакторе. Оба обходят `[]FieldDescriptor`, который объект отдаёт
сам.

## Сборка и запуск

```
cd lab3
go mod tidy
go run ./cmd/app
```

Готовая сборка: `bin/lab3/lab3.exe`.

## Формат файла (текст, вариант 3)

```
[BEGIN Car]
ID=v-001
Manufacturer=Toyota
Model=Corolla
Year=2020
MaxSpeedKmh=180
WheelCount=4
NumDoors=4
HasAirCon=true
BodyStyle=Sedan
[END]

[BEGIN Helicopter]
...
[END]
```

Переводы строк, возвраты каретки и обратные слеши внутри строковых
полей экранируются последовательностями `\n` / `\r` / `\\`, поэтому
любое значение проходит круг сериализация→десериализация без потерь.

## Возможности интерфейса

* Список загруженных объектов
* `Add` — выбор зарегистрированного типа и заполнение полей
* `Edit` — изменение выделенного объекта
* `Delete` — удаление выделенного объекта
* `Save…` / `Load…` — стандартный диалог выбора файла
* `Load default` — быстрая кнопка для чтения `vehicles.txt` из
  рабочего каталога

## Тесты

```
go test ./internal/storage/...
```

каждого встроенного класса.
